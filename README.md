# NOTEVIL//NET 🌐

A lightweight, minimalist network connection tool based on the `sing-box` core. 
Designed for easy access to unrestricted content and stable, low-latency gaming with **Hysteria2** support.

## 🚀 Key Features
- **Hysteria2 Support**: Optimized out-of-the-box for high-performance and reliable connections.
- **Gaming Stability**: Built-in **gVisor** stack and MTU optimization specifically for competitive games like **PUBG**.
- **User-Friendly Interface**: Clean and intuitive UI for easy connection management.
- **Wintun Integration**: Automatic setup of a TUN interface for complete system traffic proxying (requires Administrator privileges).
- **Import Support**: (Coming soon) Paste VLESS/Vmess links to instantly configure the outbound.

## 📦 Tech Stack
- **Python 3.x**
- **PyQt6**: UI framework.
- **sing-box**: High-performance universal proxy core.
- **gVisor**: A user-space network stack for stability and isolation.

## ⚙️ Installation
For the easiest experience, download and run the latest installer from the [Releases](https://github.com/notevil076/notevil-net-v2/releases) section.

## 🛡️ Requirements
- Windows 10/11
- Administrator privileges for network driver setup.
- NVIDIA GPU (optional, for monitoring data if added later).
